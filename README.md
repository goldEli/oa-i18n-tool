# ifun-oa-i18n 

ifun oa 国际化工具

## Features

* 自动生成国际化资源
* 自动生成国际化代码

## Extension Settings

配置国际化资源路径(绝对路径)

```json
{
 "ifun-oa-i18n.url": {
    "enPath": "D:\\projects\\web-react\\src\\locals\\en.json",
    "zhPath": "D:\\projects\\web-react\\src\\locals\\zh.json"
  }
}
```
